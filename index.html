What the board understands

  • New = Node(5) — creates a node
  • head = New — plants a head flag (any name works — curr, tail, anything)
  • New.next = New2 or New.next = None — wires (or clears) a link
  • curr = curr.next or curr = curr.next.next — hops a pointer forward
  • while curr is not None: — loops until a pointer runs off the end
  • print(curr.data) — logs to OUTPUT
  • The class Node: block up top is shown but never "run" — it's just there for reference.
  • Drag any pointer flag onto a different node, or a node's wire tip onto a new target, by hand.
